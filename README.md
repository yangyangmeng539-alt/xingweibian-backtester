# 形位变股票结构观察器

形位变股票结构观察器，是一个面向 A 股 / 港股的股票结构观察、历史路径回看、节点预判、关系图与产业链联动分析工具。

本项目尝试将“观其形，知其位，复察其变”的结构观察方法落地到股票市场历史数据分析中。

## 功能概览

- A 股 / 港股历史行情观察
- 历史节点路径回看
- 节点预判解释
- 关系图观察
- 产业链图观察
- 相关股票状态矩阵
- 市场态 / 指数态辅助观察
- 本地缓存与离线数据包导入

## 说明

本软件仅用于历史数据观察、结构研究与方法验证，不构成任何投资建议，不接券商交易接口，不提供自动交易功能。
